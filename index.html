🔑 Kết nối YouTube

Để "tìm bài gì cũng ra", web cần một YouTube API key miễn phí của Google. Làm 1 lần, ~2 phút.

  1. Vào Google Cloud Console → bấm Enable (bật YouTube Data API v3).
  2. Vào Credentials → Create credentials → API key.
  3. Copy chuỗi key (kiểu AIza...) rồi dán vào ô dưới.
💡 Mỗi key miễn phí cho khoảng 100 lượt tìm/ngày. Đủ dùng cá nhân. Key lưu trên máy bạn, không gửi đi đâu khác ngoài Google.
